# Installation Elementor — L’énergie des Runes

## Thèmes à utiliser

- parent : `edr-elementor`
- enfant : `edr-elementor-child`

## Installation

1. Téléverse `edr-elementor`
2. Téléverse `edr-elementor-child`
3. Active de préférence `edr-elementor-child`
4. Installe `Elementor`
5. Si tu l’as, installe `Elementor Pro` pour le Theme Builder

## Reprise en main du menu

Le plus simple ensuite :

1. aller dans `Templates > Theme Builder`
2. créer un `Header`
3. utiliser le widget `Nav Menu`
4. gérer l’ordre des liens directement dans le menu WordPress classique

Ainsi :

- le design du header se fait dans Elementor
- l’ordre des liens se fait dans `Apparence > Menus`

## Reprise en main du header

Dans Elementor Theme Builder, tu pourras modifier :

- logo
- menu
- espacement
- alignements
- CTA
- infos secondaires
- version mobile

## Ce que fait le thème

Le thème sert surtout de socle :

- palette
- typos
- styles globaux
- header/footer de secours
- compatibilité Elementor

## Ce que fait Elementor

Elementor servira pour :

- le vrai header final
- le footer final
- la page d’accueil
- les pages services
- les pages formations
- les landing pages
