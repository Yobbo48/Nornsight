# Kit WordPress - Lenergie des Runes

Ce dossier contient un point de depart concret pour la migration WordPress.

## Contenu

- `wp-content/themes/lenergiedesrunes/`: theme custom leger
- `../docs/wordpress-online-strategy.md`: strategie de refonte
- `../docs/page-copy-guide.md`: contenus de base a integrer
- `../docs/seo-redirect-map.csv`: base de travail pour les 301

## Installation rapide

1. Copier le dossier `lenergiedesrunes` dans `wp-content/themes/`
2. Activer le theme dans WordPress
3. Creer les pages avec ces slugs:
   - `guidances-runiques`
   - `soins-energetiques-en-ligne`
   - `ateliers-runes-en-ligne`
   - `a-propos`
4. Definir une page statique comme page d'accueil
5. Renseigner les coordonnees et le lien de reservation dans `Apparence > Personnaliser`
6. Creer les menus principal et footer
7. Integrer les textes de `docs/page-copy-guide.md`

## Note

Le theme a ete pense pour rester simple a administrer avec Gutenberg, sans builder lourd.
