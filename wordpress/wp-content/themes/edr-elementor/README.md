# EDR Elementor

Base WordPress classique et légère pour travailler avec Elementor.

## Principe

- thème parent simple
- compatible Elementor Theme Builder
- header/footer de secours si Elementor ne les remplace pas
- liberté maximale pour retoucher header, hero, sections et pages dans Elementor

## Usage recommandé

1. activer ce thème
2. installer Elementor
3. créer le header dans `Elementor > Theme Builder`
4. créer le footer dans `Elementor > Theme Builder`
5. construire la page d'accueil et les pages stratégiques dans Elementor

## Pourquoi cette base

Elle évite :

- la rigidité des template parts block theme
- les menus difficiles à reprendre dans l'éditeur de site
- les parties globales verrouillées ou peu intuitives

Elle garde :

- une base légère
- un design system propre
- une compatibilité WordPress simple
