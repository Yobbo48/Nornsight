# Pivot Elementor — L’énergie des Runes

## Pourquoi on pivote

Le block theme `edr-core` a montré ses limites sur ton usage réel :

- header trop rigide
- navigation peu agréable à reprendre en main
- parties globales parfois verrouillées ou peu intuitives
- sensation trop "bloc"

## Nouvelle décision

On part sur une base :

- classique
- légère
- compatible Elementor
- plus permissive pour le header, le hero et les pages

## Thèmes créés

- `/wordpress/wp-content/themes/edr-elementor/`
- `/wordpress/wp-content/themes/edr-elementor-child/`

## Logique de travail

- le thème donne le cadre visuel
- Elementor gère le header, le footer et les pages importantes
- WordPress reste propre et maintenable

## Ce que tu pourras modifier facilement

- le menu
- le header
- le footer
- les colonnes
- les espacements
- les visuels
- les CTA
- les sections entières

## Recommandation

Utiliser :

1. `EDR Elementor` comme parent
2. `EDR Elementor Child` comme enfant actif
3. `Elementor` pour les pages
4. `Theme Builder` pour header/footer
